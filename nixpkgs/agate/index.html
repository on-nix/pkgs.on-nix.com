nixpkgs / agate - Agate is a server for the Gemini network protocol, built with the Rust programming language. Agate has very few features, and can only serve static files. It uses async I/O, and should be quite efficient even when running on low-end hardware and serving many concurrent requests.

Homepage - https://proxy.vulpes.one/gemini/qwertqwefsday.eu/agate.gmi
