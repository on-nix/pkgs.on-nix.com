nixpkgs / spice_gtk - spice-gtk is a GTK 3 SPICE widget. It features glib-based objects for SPICE protocol parsing and a gtk widget for embedding the SPICE display into other applications such as virt-manager. Python bindings are available too.

Homepage - https://www.spice-space.org/
