nixpkgs / libcint - libcint is an open source library for analytical Gaussian integrals. It provides C/Fortran API to evaluate one-electron / two-electron integrals for Cartesian / real-spheric / spinor Gaussian type functions.

Homepage - http://wiki.sunqm.net/libcint
