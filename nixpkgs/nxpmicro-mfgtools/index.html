nixpkgs / nxpmicro-mfgtools - UUU (Universal Update Utility) is a command line tool, evolved out of MFGTools (aka MFGTools v3). One of the main purposes is to upload images to I.MX SoC's using at least their boot ROM. With time, the need for an update utility portable to Linux and Windows increased. UUU has the same usage on both Windows and Linux. It means the same script works on both OS.

Homepage - https://github.com/NXPmicro/mfgtools
