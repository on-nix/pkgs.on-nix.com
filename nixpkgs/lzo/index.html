nixpkgs / lzo - LZO is a portable lossless data compression library written in ANSI C. Both the source code and the compressed data format are designed to be portable across platforms. LZO offers pretty fast compression and *extremely* fast decompression. While it favours speed over compression ratio, it includes slower compression levels achieving a quite competitive compression ratio while still decompressing at this very high speed.

Homepage - http://www.oberhumer.com/opensource/lzo
