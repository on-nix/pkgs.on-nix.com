nixpkgs / usbguard-nox - USBGuard is a software framework for implementing USB device authorization policies (what kind of USB devices are authorized) as well as method of use policies (how a USB device may interact with the system). Simply put, it is a USB device whitelisting tool.

Homepage - https://usbguard.github.io/
