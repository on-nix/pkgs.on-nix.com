nixpkgs / acsccid - acsccid is a PC/SC driver for Linux/Mac OS X and it supports ACS CCID smart card readers. This library provides a PC/SC IFD handler implementation and communicates with the readers through the PC/SC Lite resource manager (pcscd). acsccid is based on ccid. See CCID free software driver for more information: https://ccid.apdu.fr/ It can be enabled in /etc/nixos/configuration.nix by adding: services.pcscd.enable = true; services.pcscd.plugins = [ pkgs.acsccid ];

Homepage - https://github.com/acshk/acsccid
