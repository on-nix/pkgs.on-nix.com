nixpkgs / whois - This package provides a commandline client for the WHOIS (RFC 3912) protocol, which queries online servers for information such as contact details for domains and IP address assignments. It can intelligently select the appropriate WHOIS server for most queries.

Homepage - https://packages.qa.debian.org/w/whois.html
