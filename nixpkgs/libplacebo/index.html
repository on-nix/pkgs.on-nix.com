nixpkgs / libplacebo - Reusable library for GPU-accelerated image/view processing primitives and shaders, as well a batteries-included, extensible, high-quality rendering pipeline (similar to mpv's vo_gpu). Supports Vulkan, OpenGL and Metal (via MoltenVK).

Homepage - https://code.videolan.org/videolan/libplacebo
