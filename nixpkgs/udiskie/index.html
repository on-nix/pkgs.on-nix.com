nixpkgs / udiskie - udiskie is a udisks2 front-end that allows to manage removeable media such as CDs or flash drives from userspace. Its features include: - automount removable media - notifications - tray icon - command line tools for manual un-/mounting - LUKS encrypted devices - unlocking with keyfiles (requires udisks 2.6.4) - loop devices (mounting iso archives) - password caching (requires python keyutils 0.3)

Homepage - https://github.com/coldfix/udiskie

License - MIT

Maintainers - Anderson Torres
