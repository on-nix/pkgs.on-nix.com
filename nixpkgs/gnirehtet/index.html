nixpkgs / gnirehtet - This project provides reverse tethering over adb for Android: it allows devices to use the internet connection of the computer they are plugged on. It does not require any root access (neither on the device nor on the computer). This relies on adb, make sure you have the required permissions/udev rules.

Homepage - https://github.com/Genymobile/gnirehtet
