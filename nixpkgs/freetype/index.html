nixpkgs / freetype - FreeType is a portable and efficient library for rendering fonts. It supports TrueType, Type 1, CFF fonts, and WOFF, PCF, FNT, BDF and PFR fonts. It has a bytecode interpreter and has an automatic hinter called autofit which can be used instead of hinting instructions included in fonts.

Homepage - https://www.freetype.org/
