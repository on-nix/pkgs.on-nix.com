nixpkgs / mcrcon - Mcrcon is a powerful Minecraft RCON terminal client with Bukkit coloring support. It is well suited for remote administration and to be used as part of automated server maintenance scripts. It does not trigger "IO: Broken pipe" or "IO: Connection reset" spam bugs on the server side.

Homepage - https://bukkit.org/threads/admin-rcon-mcrcon-remote-connection-client-for-minecraft-servers.70910/

Maintainers - Robin Stumm
