nixpkgs / libkate - This is libkate, the reference implementation of a codec for the Kate bitstream format. Kate is a karaoke and text codec meant for encapsulation in an Ogg container. It can carry Unicode text, images, and animate them.

Homepage - https://code.google.com/archive/p/libkate/
