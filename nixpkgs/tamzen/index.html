nixpkgs / tamzen - Tamzen is a monospace bitmap font. It is programatically forked from Tamsyn version 1.11, which backports glyphs from older versions while deleting deliberately empty glyphs to allow secondary/fallback fonts to provide real glyphs at those codepoints. Tamzen also has fonts that additionally provide the Powerline symbols.

Homepage - https://github.com/sunaku/tamzen-font

Maintainers - wishfort36
