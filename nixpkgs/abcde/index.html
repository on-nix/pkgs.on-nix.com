nixpkgs / abcde - abcde is a front-end command-line utility (actually, a shell script) that grabs tracks off a CD, encodes them to Ogg/Vorbis, MP3, FLAC, Ogg/Speex and/or MPP/MP+ (Musepack) format, and tags them, all in one go.

Homepage - http://abcde.einval.com/wiki/

License - GPL-2.0-or-later

Maintainers - Gabriel Ebner
