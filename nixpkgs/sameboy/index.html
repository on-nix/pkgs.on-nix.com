nixpkgs / sameboy - SameBoy is a user friendly Game Boy, Game Boy Color and Super Game Boy emulator for macOS, Windows and Unix-like platforms. SameBoy is extremely accurate and includes a wide range of powerful debugging features, making it ideal for both casual players and developers. In addition to accuracy and developer capabilities, SameBoy has all the features one would expect from an emulator – from save states to scaling filters.

Homepage - https://sameboy.github.io

License - MIT

Maintainers - Jakub Kądziołka
