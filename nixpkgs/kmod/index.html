nixpkgs / kmod - kmod is a set of tools to handle common tasks with Linux kernel modules like insert, remove, list, check properties, resolve dependencies and aliases. These tools are designed on top of libkmod, a library that is shipped with kmod.

Homepage - https://git.kernel.org/pub/scm/utils/kernel/kmod/kmod.git/
