nixpkgs / gnome-builder - Global search, auto-completion, source code map, documentation reference, and other features expected in an IDE, but with a focus on streamlining GNOME-based development projects. This package does not pull in the dependencies needed for every plugin. If you find that a plugin you wish to use doesn't work, we currently recommend running gnome-builder inside a nix-shell with appropriate dependencies loaded.

Homepage - https://wiki.gnome.org/Apps/Builder
