nixpkgs / swaylock - swaylock is a screen locking utility for Wayland compositors. Important note: If you don't use the Sway module (programs.sway.enable) you need to set "security.pam.services.swaylock = {};" manually.

Homepage - https://github.com/swaywm/swaylock
