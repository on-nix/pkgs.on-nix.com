nixpkgs / brltty - BRLTTY is a background process (daemon) which provides access to the Linux/Unix console (when in text mode) for a blind person using a refreshable braille display. It drives the braille display, and provides complete screen review functionality. Some speech capability has also been incorporated.

Homepage - https://brltty.app

Maintainers - Bram Duvigneau
