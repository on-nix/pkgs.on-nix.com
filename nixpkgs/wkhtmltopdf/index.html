nixpkgs / wkhtmltopdf - wkhtmltopdf and wkhtmltoimage are open source (LGPL) command line tools to render HTML into PDF and various image formats using the QT Webkit rendering engine. These run entirely "headless" and do not require a display or display service. There is also a C library, if you're into that kind of thing.

Homepage - https://wkhtmltopdf.org/

License - GPL-3.0-or-later

Maintainers - William Casarin
