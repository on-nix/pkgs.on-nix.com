nixpkgs / binutils - The GNU Binutils are a collection of binary tools. The main ones are `ld' (the GNU linker) and `as' (the GNU assembler). They also include the BFD (Binary File Descriptor) library, `gprof', `nm', `strip', etc.

Homepage - https://www.gnu.org/software/binutils/
