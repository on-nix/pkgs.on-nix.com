nixpkgs / boundary - Boundary provides a secure way to access hosts and critical systems without having to manage credentials or expose your network, and is entirely open source. Boundary is designed to be straightforward to understand, highly scalable, and resilient. It can run in clouds, on-prem, secure enclaves and more, and does not require an agent to be installed on every end host.
