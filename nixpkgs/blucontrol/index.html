nixpkgs / blucontrol - This application is a blue light filter, with the main focus on configurability. Configuration is done in Haskell in the style of xmonad. Blucontrol makes use of monad transformers and allows monadic calculation of gamma values and recoloring. The user chooses, what will be captured in the monadic state.

Homepage - https://github.com/jumper149/blucontrol
