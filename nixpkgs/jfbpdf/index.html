nixpkgs / jfbpdf - A very fast PDF and image viewer for the Linux framebuffer with some advanced and unique features, including: - Reads PDFs (MuPDF) and common image formats (Imlib2) - Supports arbitrary zoom (10% - 1000%) and rotation - Table of Contents (TOC) viewer for PDF documents - Multi-threaded rendering on multi-core machines - Asynchronous background rendering of the next page - Customizable multi-threaded caching
