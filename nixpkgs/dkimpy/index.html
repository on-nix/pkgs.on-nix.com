nixpkgs / dkimpy - Python module that implements DKIM (DomainKeys Identified Mail) email signing and verification. It also provides a number of conv—înient tools for command line signing and verification, as well as generating new DKIM records. This version also supports the experimental Authenticated Received Chain (ARC) protocol.
