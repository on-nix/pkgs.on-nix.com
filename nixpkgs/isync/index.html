nixpkgs / isync - mbsync (formerly isync) is a command line application which synchronizes mailboxes. Currently Maildir and IMAP4 mailboxes are supported. New messages, message deletions and flag changes can be propagated both ways.

Homepage - http://isync.sourceforge.net/

License - GPL-2.0-or-later

Maintainers - Michael Weiss - Linus Heckemann
