nixpkgs / python38Packages.eyeD3 - eyeD3 is a Python module and command line program for processing ID3 tags. Information about mp3 files (i.e bit rate, sample frequency, play time, etc.) is also provided. The formats supported are ID3 v1.0/v1.1 and v2.3/v2.4.

Homepage - https://eyed3.nicfit.net/

License - GPL-2.0

Maintainers - Jason O'Conal
