nixpkgs / pspp - PSPP is a program for statistical analysis of sampled data. It is a Free replacement for the proprietary program SPSS. PSPP can perform descriptive statistics, T-tests, anova, linear and logistic regression, cluster analysis, factor analysis, non-parametric tests and more. Its backend is designed to perform its analyses as fast as possible, regardless of the size of the input data. You can use PSPP with its graphical interface or the more traditional syntax commands.

Homepage - https://www.gnu.org/software/pspp/
