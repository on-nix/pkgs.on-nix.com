nixpkgs / honggfuzz - Honggfuzz is a security oriented, feedback-driven, evolutionary, easy-to-use fuzzer with interesting analysis options. It is multi-process and multi-threaded, blazingly fast when the persistent fuzzing mode is used and has a solid track record of uncovered security bugs. Honggfuzz uses low-level interfaces to monitor processes and it will discover and report hijacked/ignored signals from crashes. Feed it a simple corpus directory (can even be empty for the feedback-driven fuzzing), and it will work its way up, expanding it by utilizing feedback-based coverage metrics.

Homepage - https://honggfuzz.dev/
