nixpkgs / nerdfonts - Nerd Fonts is a project that attempts to patch as many developer targeted and/or used fonts as possible. The patch is to specifically add a high number of additional glyphs from popular 'iconic fonts' such as Font Awesome, Devicons, Octicons, and others.

Homepage - https://nerdfonts.com/

License - MIT

Maintainers - Doron Behar
