nixpkgs / udunits - The UDUNITS package supports units of physical quantities. Its C library provides for arithmetic manipulation of units and for conversion of numeric values between compatible units. The package contains an extensive unit database, which is in XML format and user-extendable. The package also contains a command-line utility for investigating units and converting values.

Homepage - https://www.unidata.ucar.edu/software/udunits/

License - Apache-2.0

Maintainers - Anderson Torres - Pascal Wittmann
