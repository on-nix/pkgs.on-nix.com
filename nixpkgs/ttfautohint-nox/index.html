nixpkgs / ttfautohint-nox - A library and two programs which take a TrueType font as the input, remove its bytecode instructions (if any), and return a new font where all glyphs are bytecode hinted using the information given by FreeType’s auto-hinting module.

Homepage - https://www.freetype.org/ttfautohint
