nixpkgs / zopfli - Zopfli Compression Algorithm is a compression library programmed in C to perform very good, but slow, deflate or zlib compression. This library can only compress, not decompress. Existing zlib or deflate libraries can decompress the data.

Homepage - https://github.com/google/zopfli
