nixpkgs / wayland-protocols - wayland-protocols contains Wayland protocols that add functionality not available in the Wayland core protocol. Such protocols either add completely new functionality, or extend the functionality of some other protocol either in Wayland core, or some other protocol in wayland-protocols.

Homepage - https://gitlab.freedesktop.org/wayland/wayland-protocols
