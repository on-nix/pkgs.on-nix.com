nixpkgs / tmuxPlugins.continuum - Features: * continuous saving of tmux environment * automatic tmux start when computer/server is turned on * automatic restore when tmux is started Together, these features enable uninterrupted tmux usage. No matter the computer or server restarts, if the machine is on, tmux will be there how you left it off the last time it was used.

Homepage - https://github.com/tmux-plugins/tmux-continuum
