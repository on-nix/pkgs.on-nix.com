nixpkgs / subtitleeditor - Subtitle Editor is a GTK 3 tool to edit subtitles for GNU/Linux/*BSD. It can be used for new subtitles or as a tool to transform, edit, correct and refine existing subtitle. This program also shows sound waves, which makes it easier to synchronise subtitles to voices.

Homepage - http://kitone.github.io/subtitleeditor/

License - GPL-3.0-or-later

Maintainers - Philip Lykke Carlsen
