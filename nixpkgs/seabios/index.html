nixpkgs / seabios - SeaBIOS is an open source implementation of a 16bit X86 BIOS. It can run in an emulator or it can run natively on X86 hardware with the use of coreboot. SeaBIOS is the default BIOS for QEMU and KVM.

Homepage - http://www.seabios.org
