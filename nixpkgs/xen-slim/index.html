nixpkgs / xen-slim - Slimmed-down version of Xen that reuses nixpkgs packages as much as possible. Different parts may get out of sync, but this builds faster and uses less space than vanilla. Use with `qemu_xen` from nixpkgs. Includes: * firmware/etherboot/ipxe.git: Xen's fork of iPXE. * qemu-xen-traditional: Xen's fork of upstream Qemu that uses old device model. * xen-libhvm-dir-remote: Helper library for reading ACPI and SMBIOS firmware values from the host system for use with the HVM guest firmware pass-through feature in Xen.

Homepage - http://www.xen.org/
