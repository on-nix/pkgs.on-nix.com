nixpkgs / smc - SMC (State Machine Compiler) takes a text input file describing states, events and actions of a state machine and generates source code that implements the state machine. SMC supports many target languages: C, C++, DotNet, Groovy, java, Java, JavaScript, Lua, ObjC, Perl, Php, Python, Ruby, Scala, Tcl. SMC can also generate GraphViz state diagrams from the input file.
