nixpkgs / tecoc - For those who don't know: TECO is the acronym of Tape Editor and COrrector (because it was a paper tape edition tool in its debut days). Now the acronym follows after Text Editor and Corrector, or Text Editor Character-Oriented. TECO is a character-oriented text editor, originally developed by Dan Murphy at MIT circa 1962. It is also a Turing-complete imperative interpreted programming language for text manipulation, done via user-loaded sets of macros. In fact, the venerable Emacs was born as a set of Editor MACroS for TECO. TECOC is a portable C implementation of TECO-11.

Homepage - https://github.com/blakemcbride/TECOC
