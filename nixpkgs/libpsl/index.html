nixpkgs / libpsl - libpsl is a C library for the Publix Suffix List (PSL). A "public suffix" is a domain name under which Internet users can directly register own names. Browsers and other web clients can use it to avoid privacy-leaking "supercookies" and "super domain" certificates, for highlighting parts of the domain in a user interface or sorting domain lists by site.
