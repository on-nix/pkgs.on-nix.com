nixpkgs / dav1d - The goal of this project is to provide a decoder for most platforms, and achieve the highest speed possible to overcome the temporary lack of AV1 hardware decoder. It supports all features from AV1, including all subsampling and bit-depth parameters.

Homepage - https://code.videolan.org/videolan/dav1d/
