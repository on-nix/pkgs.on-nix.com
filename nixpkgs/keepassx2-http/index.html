nixpkgs / keepassx2-http - A community fork of KeePassX, which is itself a port of KeePass Password Safe. The goal is to extend and improve KeePassX with new features and bugfixes, to provide a feature-rich, fully cross-platform and modern open-source password manager. Accessible via native cross-platform GUI, CLI, has browser integration using the KeePassXC Browser Extension (https://github.com/keepassxreboot/keepassxc-browser)
