nixpkgs / python38Packages.atpublic - This is a very simple decorator and function which populates a module's __all__ and optionally the module globals. This provides both a pure-Python implementation and a C implementation. It is proposed that the C implementation be added to builtins_ for Python 3.6. This proposal seems to have been rejected, for more information see https://bugs.python.org/issue26632.
