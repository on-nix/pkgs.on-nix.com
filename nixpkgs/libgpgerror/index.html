nixpkgs / libgpgerror - Libgpg-error is a small library that defines common error values for all GnuPG components. Among these are GPG, GPGSM, GPGME, GPG-Agent, libgcrypt, Libksba, DirMngr, Pinentry, SmartCard Daemon and possibly more in the future.

Homepage - https://www.gnupg.org/software/libgpg-error/index.html
