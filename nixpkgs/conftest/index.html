nixpkgs / conftest - Conftest helps you write tests against structured configuration data. Using Conftest you can write tests for your Kubernetes configuration, Tekton pipeline definitions, Terraform code, Serverless configs or any other config files. Conftest uses the Rego language from Open Policy Agent for writing the assertions. You can read more about Rego in 'How do I write policies' in the Open Policy Agent documentation.

Homepage - https://www.conftest.dev
