nixpkgs / yandex-disk - Yandex.Disk console client for Linux lets you manage files on Disk without using a window interface or programs that support WebDAV. The advantages of the console client compared to a WebDAV connection: * low system resource requirements; * faster file reading and writing speeds; * faster syncing with Disk's server; * no need to be constantly connected to work with files.

Homepage - https://help.yandex.com/disk/cli-clients.xml

Maintainers - Sergey Mironov - Arseniy Seroka
