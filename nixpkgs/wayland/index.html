nixpkgs / wayland - Wayland is a project to define a protocol for a compositor to talk to its clients as well as a library implementation of the protocol. The wayland protocol is essentially only about input handling and buffer management, but also handles drag and drop, selections, window management and other interactions that must go through the compositor (but not rendering).

Homepage - https://wayland.freedesktop.org/

License - MIT

Maintainers - Michael Weiss - Cody Opel - Alyssa Ross
