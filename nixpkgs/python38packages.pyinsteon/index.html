nixpkgs / python38Packages.pyinsteon - This is a Python package to interface with an Insteon Modem. It has been tested to work with most USB or RS-232 serial based devices such as the 2413U, 2412S, 2448A7 and Hub models 2242 and 2245.

Homepage - https://github.com/pyinsteon/pyinsteon

Maintainers - Fabian Affolter
