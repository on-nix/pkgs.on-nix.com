nixpkgs / numix-solarized-gtk-theme - This is a fork of the Numix GTK theme that replaces the colors of the theme and icons to use the solarized theme with a solarized green accent color. This theme supports both the dark and light theme, just as Numix proper.

Homepage - https://github.com/Ferdi265/numix-solarized-gtk-theme
