nixpkgs / iw - iw is a new nl80211 based CLI configuration utility for wireless devices. It supports all new drivers that have been added to the kernel recently. The old tool iwconfig, which uses Wireless Extensions interface, is deprecated and it's strongly recommended to switch to iw and nl80211.

Homepage - https://wireless.wiki.kernel.org/en/users/Documentation/iw
