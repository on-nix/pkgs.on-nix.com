nixpkgs / fsuae - FS-UAE integrates the most accurate Amiga emulation code available from WinUAE. FS-UAE emulates A500, A500+, A600, A1200, A1000, A3000 and A4000 models, but you can tweak the hardware configuration and create customized Amigas.

Homepage - https://fs-uae.net

License - GPL-2.0-or-later

Maintainers - Anderson Torres
