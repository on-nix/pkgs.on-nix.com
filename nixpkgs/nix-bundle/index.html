nixpkgs / nix-bundle - nix-bundle is a way to package Nix attributes into single-file executables. Benefits: - Single-file output - Can be run by non-root users - No runtime - Distro agnostic - No installation

Homepage - https://github.com/matthewbauer/nix-bundle
