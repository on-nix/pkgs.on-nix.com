nixpkgs / tikzit - TikZiT is a simple GUI editor for graphs and string diagrams. Its native file format is a subset of PGF/TikZ, which means TikZiT files can be included directly in papers typeset using LaTeX. For preview support the texlive package 'preview' has to be installed.
