nixpkgs / libcdio - GNU libcdio is a library for OS-independent CD-ROM and CD image access. It includes a library for working with ISO-9660 filesystems (libiso9660), as well as utility programs such as an audio CD player and an extractor.

Homepage - https://www.gnu.org/software/libcdio/
