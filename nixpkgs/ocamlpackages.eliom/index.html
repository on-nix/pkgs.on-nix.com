nixpkgs / ocamlPackages.eliom - Eliom is a framework for programming Web sites and client/server Web applications. It introduces new concepts to simplify programming common behaviours and uses advanced static typing features of OCaml to check many properties of the Web site at compile time. If you want to write a Web application, Eliom makes possible to write the whole application as a single program (client and server parts). A syntax extension is used to distinguish both parts and the client side is compiled to JS using Ocsigen Js_of_ocaml.

Homepage - http://ocsigen.org/eliom/
