nixpkgs / php80Packages.phpstan - PHPStan focuses on finding errors in your code without actually running it. It catches whole classes of bugs even before you write tests for the code. It moves PHP closer to compiled languages in the sense that the correctness of each line of the code can be checked before you run the actual line.

Homepage - https://github.com/phpstan/phpstan
