nixpkgs / tmuxPlugins.resurrect - This plugin goes to great lengths to save and restore all the details from your tmux environment. Here's what's been taken care of: * all sessions, windows, panes and their order * current working directory for each pane * exact pane layouts within windows (even when zoomed) * active and alternative session * active and alternative window for each session * windows with focus * active pane for each window * "grouped sessions" (useful feature when using tmux with multiple monitors) * programs running within a pane! More details in the restoring programs doc. Optional: * restoring vim and neovim sessions * restoring pane contents

Homepage - https://github.com/tmux-plugins/tmux-resurrect
