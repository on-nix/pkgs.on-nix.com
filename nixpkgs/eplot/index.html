nixpkgs / eplot - eplot ("easy gnuplot") is a ruby script which allows to pipe data easily through gnuplot and create plots quickly, which can be saved in postscript, PDF, PNG or EMF files. Plotting of multiple files into a single diagram is supported. This package also includes the complementary 'ec' tool (say "extract column").

Homepage - https://perso.liris.cnrs.fr/christian.wolf/software/eplot/
