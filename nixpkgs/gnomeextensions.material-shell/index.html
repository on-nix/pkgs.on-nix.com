nixpkgs / gnomeExtensions.material-shell - A modern desktop interface for Linux - packaged as an extension for GNOME Shell. Improve your user experience and get rid of the anarchy of traditional desktop workflows. Designed to simplify navigation and reduce the need to manipulate windows in order to improve productivity. It's meant to be 100% predictable and bring the benefits of tools coveted by professionals to everyone.

Homepage - https://extensions.gnome.org/extension/3357/material-shell/

License - GPL-2.0-or-later

Maintainers - piegames
