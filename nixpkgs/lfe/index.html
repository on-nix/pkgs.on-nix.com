nixpkgs / lfe - LFE, Lisp Flavoured Erlang, is a lisp syntax front-end to the Erlang compiler. Code produced with it is compatible with "normal" Erlang code. An LFE evaluator and shell is also included.

Homepage - http://lfe.io

License - Apache-2.0

Maintainers - Justin Wood - Oleksii Filonenko - Thomas Depierre - Gleb Peregud - Raphael Megzari - Eric Bailey
