nixpkgs / radamsa - Radamsa is a general purpose data fuzzer. It reads data from given sample files, or standard input if none are given, and outputs modified data. It is usually used to generate malformed data for testing programs.

Homepage - https://gitlab.com/akihe/radamsa

Maintainers - Markus Wotringer
