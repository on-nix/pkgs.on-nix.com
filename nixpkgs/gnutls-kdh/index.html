nixpkgs / gnutls-kdh - The ARPA2 project aims to add security. This is an enhanced version of GnuTLS, a project that aims to develop a library which provides a secure layer, over a reliable transport layer. It adds TLS-KDH ciphers: Kerberos + Diffie-Hellman.
