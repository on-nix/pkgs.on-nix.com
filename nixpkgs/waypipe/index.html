nixpkgs / waypipe - waypipe is a proxy for Wayland clients. It forwards Wayland messages and serializes changes to shared memory buffers over a single socket. This makes application forwarding similar to ssh -X feasible.

Homepage - https://mstoeckl.com/notes/gsoc/blog.html
