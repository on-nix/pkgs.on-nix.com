nixpkgs / higan - higan is a multi-system emulator, originally developed by Near, with an uncompromising focus on accuracy and code readability. It currently emulates the following systems: Famicom, Famicom Disk System, Super Famicom, Super Game Boy, Game Boy, Game Boy Color, Game Boy Advance, Game Boy Player, SG-1000, SC-3000, Master System, Game Gear, Mega Drive, Mega CD, PC Engine, SuperGrafx, MSX, MSX2, ColecoVision, Neo Geo Pocket, Neo Geo Pocket Color, WonderSwan, WonderSwan Color, SwanCrystal, Pocket Challenge V2.

Homepage - https://github.com/higan-emu/higan

Maintainers - Anderson Torres
