nixpkgs / pax-utils - A suite of ELF tools to aid auditing systems. Contains various ELF related utils for ELF32, ELF64 binaries useful for displaying PaX and security info on a large groups of binary files.

Homepage - https://wiki.gentoo.org/wiki/Hardened/PaX_Utilities
