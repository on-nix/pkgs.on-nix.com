nixpkgs / yadm - yadm is a dotfile management tool with 3 main features: * Manages files across systems using a single Git repository. * Provides a way to use alternate files on a specific OS or host. * Supplies a method of encrypting confidential data so it can safely be stored in your repository.

Homepage - https://github.com/TheLocehiliosan/yadm
