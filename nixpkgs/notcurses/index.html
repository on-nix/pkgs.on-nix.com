nixpkgs / notcurses - A library facilitating complex TUIs on modern terminal emulators, supporting vivid colors, multimedia, and Unicode to the maximum degree possible. Things can be done with Notcurses that simply can't be done with NCURSES. It is not a source-compatible X/Open Curses implementation, nor a replacement for NCURSES on existing systems.

Homepage - https://github.com/dankamongmen/notcurses
