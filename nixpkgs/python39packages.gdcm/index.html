nixpkgs / python39Packages.gdcm - Grassroots DICOM (GDCM) is an implementation of the DICOM standard designed to be open source so that researchers may access clinical data directly. GDCM includes a file format definition and a network communications protocol, both of which should be extended to provide a full set of tools for a researcher or small medical imaging vendor to interface with an existing medical database.

Homepage - http://gdcm.sourceforge.net/

Maintainers - Thiago Franco de Moraes
