nixpkgs / phantomjs - PhantomJS is a headless WebKit with JavaScript API. It has fast and native support for various web standards: DOM handling, CSS selector, JSON, Canvas, and SVG. PhantomJS is an optimal solution for: - Headless Website Testing - Screen Capture - Page Automation - Network Monitoring

Homepage - https://phantomjs.org/
