nixpkgs / gildas - GILDAS is a collection of state-of-the-art software oriented toward (sub-)millimeter radioastronomical applications (either single-dish or interferometer). It is daily used to reduce all data acquired with the IRAM 30M telescope and Plateau de Bure Interferometer PDBI (except VLBI observations). GILDAS is easily extensible. GILDAS is written in Fortran-90, with a few parts in C/C++ (mainly keyboard interaction, plotting, widgets).

Homepage - http://www.iram.fr/IRAMFR/GILDAS/gildas.html
