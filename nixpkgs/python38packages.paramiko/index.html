nixpkgs / python38Packages.paramiko - This is a library for making SSH2 connections (client or server). Emphasis is on using SSH2 as an alternative to SSL for making secure connections between python scripts. All major ciphers and hash methods are supported. SFTP client and server mode are both supported too.

Homepage - https://github.com/paramiko/paramiko/
