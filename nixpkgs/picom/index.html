nixpkgs / picom - A fork of XCompMgr, which is a sample compositing manager for X servers supporting the XFIXES, DAMAGE, RENDER, and COMPOSITE extensions. It enables basic eye-candy effects. This fork adds additional features, such as additional effects, and a fork at a well-defined and proper place. The package can be installed in debug mode as: picom.override { withDebug = true; } For gdb to find the source files, you need to run gdb in the bin directory of picom package in the nix store.

Homepage - https://github.com/yshui/picom
