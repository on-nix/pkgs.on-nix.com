nixpkgs / libxkbcommon - libxkbcommon is a keyboard keymap compiler and support library which processes a reduced subset of keymaps as defined by the XKB (X Keyboard Extension) specification. It also contains a module for handling Compose and dead keys.

Homepage - https://xkbcommon.org

License - MIT

Maintainers - Michael Weiss - Thomas Tuegel
