nixpkgs / simgrid - SimGrid is a toolkit that provides core functionalities for the simulation of distributed applications in heterogeneous distributed environments. The specific goal of the project is to facilitate research in the area of distributed and parallel application scheduling on distributed computing platforms ranging from simple network of workstations to Computational Grids.

Homepage - https://simgrid.org/

License - LGPL-2.0-or-later

Maintainers - Michael Mercier - Millian Poquet
