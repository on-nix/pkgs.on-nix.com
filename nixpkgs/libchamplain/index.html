nixpkgs / libchamplain - libchamplain is a C library providing a ClutterActor to display maps. It also provides a GTK widget to display maps in GTK applications. Python and Perl bindings are also available. It supports numerous free map sources such as OpenStreetMap, OpenCycleMap, OpenAerialMap, and Maps for free.

Homepage - https://wiki.gnome.org/Projects/libchamplain

License - LGPL-2.0-or-later

Maintainers - Tor Hedin Brønner - Jan Tojnar - Daniel Șerbănescu - Maxine Aubrey
