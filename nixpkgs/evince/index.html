nixpkgs / evince - Evince is a document viewer for multiple document formats. It currently supports PDF, PostScript, DjVu, TIFF and DVI. The goal of Evince is to replace the multiple document viewers that exist on the GNOME Desktop with a single simple application.

Homepage - https://wiki.gnome.org/Apps/Evince

License - GPL-2.0-or-later
