nixpkgs / homesick - Homesick is sorta like rip, but for dotfiles. It uses git to clone a repository containing dotfiles, and saves them in ~/.homesick. It then allows you to symlink all the dotfiles into place with a single command.

Homepage - https://github.com/technicalpickles/homesick

License - MIT

Maintainers - Aaron Schif - Nick Novitski
