nixpkgs / ocamlPackages.vg - Vg is an OCaml module for declarative 2D vector graphics. In Vg, images are values that denote functions mapping points of the cartesian plane to colors. The module provides combinators to define and compose these values. Renderers for PDF, SVG and the HTML canvas are distributed with the module. An API allows to implement new renderers.

Homepage - https://erratique.ch/software/vg
