nixpkgs / nitrogen - nitrogen is a lightweight utility that can set the root background on X11. It operates independently of any desktop environment, and supports multi-head with Xinerama. Wallpapers are browsable with a convenient GUI, and settings are stored in a human-readable config file.

Homepage - https://github.com/l3ib/nitrogen
