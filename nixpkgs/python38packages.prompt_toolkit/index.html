nixpkgs / python38Packages.prompt_toolkit - prompt_toolkit could be a replacement for readline, but it can be much more than that. It is cross-platform, everything that you build with it should run fine on both Unix and Windows systems. Also ships with a nice interactive Python shell (called ptpython) built on top.

Homepage - https://github.com/jonathanslenders/python-prompt-toolkit
